[2차시] 디지털 수사 장부 완성하기

수작업은 이제 그만!

본부에 제보가 쏟아지고 있어요! 하지만 1차시에서 만든 변수는 정보를 **딱 하나**만 기억할 수 있었죠. 새로운 제보를 넣으면 예전 정보가 지워지는 문제를 해결하기 위해, 말만 하면 자동으로 기록되는 **[디지털 수사 기록기]**를 만들어 봅시다!

실습 1: 1차시 수사 본부 불러오기

  • ⬜ 1. 엔트리 접속: playentry.org 로그인
  • ⬜ 2. 작품 열기: 나의 작품 중 [1단계_수집기_본인이름] 열기
  • ⬜ 3. 상태 확인: 1차시에서 만든 변수 3개와 리스트 1개가 있는지 확인!

데이터 탐정의 자동화 논리

  • 1 사건 해결을 위해 정보를 수집하고 분석해야 해요!
  • 2 분석을 잘 하려면 정보를 체계적으로 정리해서 입력해야 하죠.
  • 3 컴퓨터는 정보를 '변수'라는 그릇에 담아서 기억해요.
  • 4 변수는 정보를 딱 '한 개'만 기억할 수 있어 예전 기록이 사라져요.
  • 5 그래서 모든 제보를 사라지지 않게 쌓아둘 '리스트'라는 장부가 필요해요!
  • 🚀 자, 어떤 순서로 코딩할지 알고리즘을 설계해 봅시다!

실습 2: 디지털 기록기 코딩하기

👮 미션 1: 주인공 탐정 코딩 (현장 지휘)

오브젝트 목록에서 탐정 캐릭터를 선택하고 아래 블록을 조립하세요.

1. 장면이 시작되었을 때 시작 블록 가져오기
2. "지금부터 커피 향기 도난 사건 수사를 시작한다!"라고 2초간 말하기
🤖 미션 2: 조수 로봇 코딩 (데이터 수집)

오브젝트 목록에서 조수 로봇을 선택하고 9단계 알고리즘을 완성하세요.

1. 오브젝트를 클릭했을 때 시작 블록 가져오기
2. "언제 발생했나요?"를 묻고 대답 기다리기
3. [시간] 변수를 (대답)으로 정하기
4. "어디서 발생했나요?"를 묻고 대답 기다리기
5. [장소] 변수를 (대답)으로 정하기
6. "냄새가 얼마나 심했나요?(1~10)"를 묻고 대답 기다리기
7. [냄새 농도] 변수를 (대답)으로 정하기
8. 핵심! (시간 + 장소 + 냄새 농도) 항목을 [수사로그]에 추가하기
9. "기록 완료! 탐정님, 장부를 확인해 주세요."라고 말하기
💡 꿀팁: 8번 단계에서 3개의 값을 합칠 때는 [계산] 꾸러미의 (안녕)과 (엔트리)를 합치기 블록을 여러 개 겹쳐서 사용하세요!
🧹 미션 3: 초기화 버튼 코딩 (장부 정리)

리스트를 비우기 위해서는 반복문을 사용해야 합니다. 버튼 오브젝트에 아래 로직을 조립하세요.

1. 오브젝트를 클릭했을 때 시작 블록 가져오기
2. [흐름] 꾸러미에서 (10)번 반복하기 블록 가져오기
3. 반복 횟수(10) 자리에 [자료]의 (수사로그의 항목 수) 블록 넣기
4. 반복문 안쪽에 [자료]의 (1)번째 항목을 [수사로그]에서 삭제하기 넣기
5. "수사 장부를 깨끗이 비웠습니다!"라고 1초간 말하기
  • ⬜ 1. 작동 확인: 로봇을 클릭해 제보 데이터가 리스트에 차곡차곡 쌓이나요?
  • ⬜ 2. 초기화 확인: 버튼을 눌렀을 때 수사로그가 싹 비워지나요?
  • ⬜ 3. 최종 저장: 완성된 자동 수사 시스템을 엔트리에 저장하기!

🏅 데이터 탐정 자격 시험 (2단계)

Q1. 묻고 답하기 블록을 통해 입력받은 값을 가지고 있는 블록은?

Q2. 리스트에 데이터를 추가하면, 기존 데이터는 어떻게 되나요?

Q3. 엔트리에서 리스트를 싹 비우기 위해 필요한 알고리즘은 무엇인가요?

Q4. 여러 개의 변수 값을 하나로 합쳐서 보여주고 싶을 때 사용하는 꾸러미는?

Q5. 데이터 수집기를 만든 탐정단이 다음에 해야 할 일은?
